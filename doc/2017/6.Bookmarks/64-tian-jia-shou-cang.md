## 添加收藏

请求地址：
`POST https://api.cnblogs.com/api/Bookmarks`



请求参数头：

|参数名|参数值|
|:---:|:---:|
|Authorization|Bearer {token}|
|Content-Type|application/json|

请求参数示例：
```
{
  "WzLinkId": 1,
  "Title": "sample string 2",
  "LinkUrl": "sample string 3",
  "Summary": "sample string 4",
  "Tags": [
    "sample string 1",
    "sample string 2"
  ]
}
```
